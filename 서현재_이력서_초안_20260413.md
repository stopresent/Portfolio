## 5. 프로젝트

### TheSword
- 기간: 2024.05 -
- 형태: JRPG 스타일 1인 전략 게임 / 4인 팀 프로젝트
- 사용 기술: Unity, C#, Git, GitHub Desktop
- 담당 업무:
  - Unity `EditorWindow` 기반 맵 자동 생성 툴 구현
  - 전투 게임 로직 구현
  - UI 자동화 구조 구현
  - CSV/JSON 기반 데이터 연동
  - 다국어 처리 구조 구현
- 주요 성과:
  - 수동 맵 제작의 비효율을 개선하기 위해 커스텀 맵 에디터를 구현했고, 맵 제작 시간을 60% 이상 단축
  - 팩토리 패턴과 전략 패턴을 적용해 몬스터 특성 시스템을 구현하고 기존 코드 수정 없이 특성 확장이 가능하도록 구성
  - 카메라 워킹, 쉐이크, 파티클 연출, DOTween 기반 UI 애니메이션으로 플레이 몰입감 개선
  - 코어 시스템, UI, 연출 순으로 우선순위를 조정하며 출시 가능한 수준까지 팀 개발 흐름 정리
  - itch.io 공개 경험 보유

## 8. 자기소개

Unity와 C#을 중심으로 게임 클라이언트 개발 역량을 쌓아온 지원자입니다. 다양한 팀 프로젝트를 통해 UI, 전투 로직, 데이터 연동, 해상도 대응, 네트워크 연동 문제를 직접 구현하고 개선해 왔습니다. 특히 `TheSword` 프로젝트에서는 맵 자동 생성 툴과 UI 자동화 구조를 구현해 개발 효율과 유지보수성을 함께 높였고, `MessMath`에서는 멀티플레이 환경에서 발생한 해상도 동기화 문제를 해결하며 실제 문제 해결 역량을 키웠습니다.

또한 알고리즘 학습을 꾸준히 병행하며 `solved.ac Platinum 5`에 도달했고, DirectX12와 게임 수학 등 기초 체력도 함께 쌓아왔습니다. 실제로 게임을 기획부터 구현, 테스트, 공개까지 연결해 본 경험을 바탕으로, 사용자 경험에 직접 영향을 주는 게임 클라이언트 개발자로 기여하겠습니다.
